Package com.github.tagc.semver.projectstrategy

  • Interface Summary 
    Interface Description
    ProjectVersionSelectorStrategy Handles the selection of an instance of Version to set for the version of the project that this plugin is applied to.
  • Class Summary 
    Class Description
    AbstractProjectVersionSelectorStrategy An abstract class that provides default behaviour for concrete instances of ProjectVersionSelectorStrategy.
    DefaultVersionSelectorStrategy Handles version selection for projects if no other ProjectVersionSelectorStrategy can.
    HotfixReleaseSelectorStrategy Handles version selection for projects that are on a hotfix or release Git branch.
    MasterVersionSelectorStrategy Handles version selection for projects that are on the master Git branch.
    VersionSelectionData A class for producing immutable objects that capture the information required by instances of ProjectVersionSelectionStrategy to choose which Version to select for the project that this plugin is applied to.
    VersionSelectionManager Manages the selection of an instance of Version to set as the version of the project that this plugin is applied to by delegating the responsibility to a chain of ProjectVersionSelectionStrategy instances.
